## Validator helper slices

```text
r1_required_blocks.py:
  checks required KDSL_RESULT fields

r1_rt_basis.py:
  checks RT:v basis wording in RT/VERIFY/S fields
  avoids treating unrelated explanatory notes as RT:v basis

r1_authority_guard.py:
  checks NEXT proposal-only and COMMIT actual/proposed/none shape
  supports same-line values and simple multi-line block fields

kdsl_template_refs.py:
  checks known template references and template safety gates

kdsl_template_expansion.py:
  checks expansion/inheritance evidence and required section groups
  does not prove full template expansion

kdsl_validate.py:
  wraps checks by target: r1 / prompt / all

run_samples.py:
  checks sample expected exit codes
```
